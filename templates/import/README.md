# قوالب الاستيراد - Import Templates

هذا المجلد يحتوي على قوالب Excel/CSV الجاهزة لاستيراد البيانات.

## الملفات المتوفرة

| القالب | الوصف |
|--------|-------|
| `schools.template_v1.xlsx` | بيانات المدارس |
| `classes.template_v1.xlsx` | بيانات الفصول الدراسية |
| `teachers.template_v1.xlsx` | بيانات المعلمين |
| `students_guardians.template_v1.xlsx` | بيانات الطلاب وأولياء الأمور |
| `subjects.template_v1.xlsx` | بيانات المواد الدراسية |

## بنية القالب

كل ملف يحتوي على:

1. **الصف الأول**: عناوين الأعمدة (Headers)
2. **الصف الثاني**: تعليمات وتلميحات لكل حقل
3. **الصف الثالث**: مثال توضيحي
4. **الصفوف 4+**: فارغة للإدخال

## ترتيب الاستيراد المطلوب

يجب استيراد الملفات بالترتيب التالي لتجنب أخطاء المراجع:

1. المدارس (schools) ← أولاً
2. الفصول (classes) ← يحتاج school_code
3. المعلمين (teachers) ← يحتاج school_code
4. الطلاب (students) ← يحتاج class_code
5. المواد (subjects) ← يحتاج teacher_code (اختياري)

## إنشاء قوالب جديدة

```bash
node scripts/generate_templates.js
```

## إصدار القالب

- الإصدار الحالي: `v1`
- عمود `_template_version` يحدد إصدار القالب
- النظام يتحقق من توافق الإصدار عند الرفع

## ملاحظات

- ملفات CSV تحتوي على BOM للتوافق مع Excel العربي
- لا تحذف صف التعليمات (الصف الثاني)
- استخدم القيم المرجعية من أوراق `_codes` عند توفرها
